参考docker安装笔记

查询docker nginx镜像

docker search nginx

拉取docker nginx镜像

docker pull nginx:latest

查看已拉取镜像

docker images

运行容器

docker run –name nginx-test -p 8080:80 -d nginx

–name nginx-test：容器名称。
-p 8080:80： 端口进行映射，将本地 8080 端口映射到容器内部的 80 端口。
-d nginx： 设置容器在在后台一直运行。

apt-get install -y iproute2

尾调用优化

Posted on 2021-02-09 | Edited on 2021-02-11 | Comments:

  • 尾调用
    在某个函数的最后一步调用其他函数，被称为尾调用。

  • 尾调用优化
    指的是在函数内部的函数不再使用上一个函数中的变量，
    如果有必须要使用的变量则改为参数的形式传入

  • 尾调用优化逻辑为: 一个函数调用另一个函数后，如果在其内部引用了上一个函数的变量，则在该函数未执行完毕时，会保留上一个函数中的变量，从而占据了内存。

  • 用途: 进行尾调用优化后，就可以做到执行时调用帧只有一项，大大节省内存
