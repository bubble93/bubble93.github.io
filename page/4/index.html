ie盒模型和google盒模型区别

Posted on 2019-05-31 | In 前端 | Comments:

IE 盒子模型的 content 部分包含了 border 和 padding。这意味着当宽高固定时增加border和padding会压缩内容的显示，而不会扩大盒子。
好处是：当想要给固定好的布局中增加border和padding的面积不会改变原有布局
可通过css3新属性box-sizing来设置
box-sizing: content-box|border-box|inherit;
简单来说就是border-box为ie盒模型，content-box为正常盒模型，inherit为继承父元素的box-sizing值

MAC空格键和回车键失效问题

Posted on 2019-05-31 | Edited on 2019-06-03 | In MAC | Comments:

问题描述：空格键和回车键失效，选中输入法，选择显示虚拟键盘，按下空格以及回车后键盘不会变灰（变灰表示有反应），按下fn后再点击空格以及回车会有反应

解决方式1：按下电源键开机的同时，马上按住option + command + P + R 持续30秒 然后解决
解决方式2：关机状态下，按下control + option + shift + 电源键 持续10秒
ps: 需要接入电源再执行1,2 解决方式. 未接入电源情况下 1 方式失效.

百度资料
OPTION+CMD+R+P+POWER：重置NVRAM 或 PRAM
Mac 会将某些设置储存在特殊内存区域中，而且即使关机这些设置也不会丢失（除非存在电池问题）。 在基于 Intel 的 Mac 上，存储位置是称为 NVRAM 的内存；而在基于 PowerPC 的 Mac 上，存储位置则是称为 PRAM 的内存。操作步骤为：
关闭电脑。
在键盘上找到以下按键：Command、Option、P 和 R。您需要在步骤 4 中同时按下这些键。
启动电脑。
按住 Command-Option-P-R 键。必须在出现灰屏前按下此组合键。
按住上述键，直到电脑重新启动，您会再次听到启动声。
松开这些键。

使用Hexo创建个人博客以及使用自定义域名(1)

Posted on 2019-05-31 | Edited on 2019-06-03 | In 博客建站 | Comments:

本教程分为3部分，1.本地部署hexo博客。2.上传本地hexo博客到github并使用github.io访问。3.配置自定义域名到hexo博客

1.本地部署hexo博客

  • 下载Node.js

  • 使用npm安装Hexo

    npm install hexo-cli -g

  • 在需要创建博客的文件夹下输入如下命令，初始化博客文件夹

    hexo init blog

  • 进入blog文件夹

    cd blog

  • 初始化所插件

    npm install

  • 在本地运行服务

    hexo server
    默认地址为：localhost:4000
