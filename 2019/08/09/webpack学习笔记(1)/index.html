开发

  • 使用 source map

    devtool: 'inline-source-map' 定位错误到源文件上，而不是编译后的文件

  • 选择一个开发工具

    为了方便在代码修改后能够实时产生效果，而不是运行npm run build。
    webpack’s Watch Mode
    webpack-dev-server
    webpack-dev-middleware

  • 使用观察者模式

    使用webpack的watch，更改完代码以后需要刷新浏览器

  • 使用 webpack-dev-server

    生成后的文件直接通过localhost:8080访问，并且修改代码保存后，自动更新

  • 使用 webpack-dev-middleware（作用尚不明确，大概是将生成后的代码直接放到一个指定位置，而非生成在dist目录下）

  • 其他代码和框架

    • React Hot Loader：实时调整 react 组件。
    • Vue Loader：此 loader 支持用于 vue 组件的 HMR，提供开箱即用体验。
    • Elm Hot Loader：支持用于 Elm 程序语言的 HMR。
    • Redux HMR：无需 loader 或插件！只需对 main store 文件进行简单的修改。
