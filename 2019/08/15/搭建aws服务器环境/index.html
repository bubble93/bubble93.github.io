安装git，nodejs，npm

  • 安装git

    sudo apt-get install git

  • 安装node

    sudo git clone https://github.com/nodejs/node.git

  • 赋予权限

    sudo chmod -R 755 node

  • 执行安装

    cd node
    sudo ./configure

  • 报错

  • 安装python

    sudo apt-get install python

  • 执行安装

    sudo ./configure

  • 报错

  • 安装gcc

    sudo apt-get install build-essential

  • 执行安装

    sudo ./configure
    sudo make //在该步骤安装许久未结束。安装失败
    sudo make install

  • 查看版本

    node --version
